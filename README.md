# Stage 1b - Profile Card

A simple web application displaying a profile card with user information, hobbies, dislikes, current epoch time, and social media links.

## Features

- **Profile Display**: Shows user avatar, name, and bio
- **Dynamic Time**: Displays current epoch time in milliseconds
- **Lists**: Hobbies and dislikes sections
- **Social Links**: Links to Twitter, LinkedIn, and GitHub
- **Responsive Design**: Optimized for different screen sizes
- **Accessibility**: Includes ARIA labels and semantic HTML

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## Setup and Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/your-repo-name.git
   ```

2. Navigate to the project directory:

   ```bash
   cd your-repo-name
   ```

3. Open `index.html` in your web browser to view the profile card.

No additional dependencies or build steps are required.

## Project Structure

- `index.html` - Main HTML file
- `style.css` - Stylesheet for the profile card
- `script.js` - JavaScript for dynamic time display
